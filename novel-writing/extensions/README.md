# novel-writing 扩展内容

本目录存放由 `/novel-learn` 分析小说后产出的技法扩展。扩展内容经用户确认后存入，是对核心模块的补充。

## 目录结构

```
extensions/
├── README.md          # 本文件
├── characters/        # 角色类型扩展（新发现的角色类型、塑造手法）
├── plot/              # 情节技法扩展（新发现的叙事手法、钩子类型）
└── prose/             # 写作技法扩展（新发现的写作技巧、对话风格）
```

## 命名规范

每个扩展文件命名格式：`[小说名]_[序号].md`

示例：
- `characters/某小说名_01.md`
- `plot/某小说名_01.md`
- `prose/某小说名_01.md`

## 与核心模块的关系

- 核心模块（`references/module-*.md`）是经过系统梳理的通用技法，稳定可靠
- 扩展内容是学习分析产出的增量技法，质量参差不齐
- 使用时应优先参考核心模块，扩展内容作为补充
- 用户可以手动删除或修改扩展内容，不影响核心模块
