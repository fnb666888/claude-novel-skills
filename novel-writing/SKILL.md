# 小说创作综合技能

基于以下小说系统梳理的创作方法论：
- 《逼我重生是吧》（幼儿园一把手 著，全书703章）— 都市重生+商业+后宫
- 《开局合欢宗，被师姐拿捏命脉》（咸鱼老白 著，全书1308章）— 玄幻修仙+后宫+幽默
- 《说好体验人生，仙子你怎么成真了》（红烧油焖虾 著，全书528章）— 玄幻修仙+百世轮回+后宫
- 《重燃青葱时代》（蜜汁姬 著，全书300+章）— 都市重生+校园+后宫

本skill包含四大模块 + 两个去耦合分类库，可独立使用也可综合使用：

**核心模块（创作技法）：**
- **模块A：角色塑造** — 创建立体、有辨识度的角色 → `references/module-a-character.md`
- **模块B：情节构造** — 设计主线、感情线、冲突线 → `references/module-b-plot.md`
- **模块C：正文写作** — 具体的写作技巧和公式 → `references/module-c-prose.md`
- **模块D：反面教材** — 常见烂文模式与写作公式 → `references/module-d-antipatterns.md`

**去耦合分类库（可自由组合的元素）：**
- **角色特点分类库** — 个性类型×情绪类型×行为动作×语言风格 → `references/character-traits.md`
- **情节元素分类库** — 事件类型×事件风格×场景氛围 → `references/plot-elements.md`

**去耦合化设计**：分类库与创作技法解耦。技法告诉你"怎么写"，分类库告诉你"有什么可选"。通过自由组合，可以创造出丰富多样的角色和情节，避免陷入单一模式。

输出规则见 → `references/output-rules.md`

## 使用方式

用户可通过以下方式调用：
- `/novel-writing 角色` — 进入角色塑造模块（加载 module-a-character.md）
- `/novel-writing 情节` — 进入情节构造模块（加载 module-b-plot.md）
- `/novel-writing 写作` — 进入正文写作模块（加载 module-c-prose.md）
- `/novel-writing 综合` — 综合使用三个模块（加载全部模块文件）
- `/novel-writing [小说文件路径]` — 分析指定小说并生成创作指南

## 去耦合化组合使用（核心设计理念）

### 设计哲学

**好小说不是按套路写出来的，而是通过自由组合创造惊喜。**

- **不是**：选一个"高冷型"女主，按套路写她永远高冷
- **而是**：高冷型×害羞情绪×咬唇行为×惜字如金语言 → 这个组合本身就是惊喜
- **不是**：选一个"误会"情节，按模板写
- **而是**：误会冲突×心动甜蜜风格×暧昧私密氛围 → 误会也能甜到牙疼

### 核心公式

```
角色表现 = 个性类型 × 情绪状态 × 行为动作 × 语言风格
章节内容 = 事件类型 × 事件风格 × 场景氛围 × 角色特点
```

### 使用流程

1. **声明核心体验类型**（暧昧/悲伤/搞笑/热血/甜蜜/治愈/...）
2. 从 `character-traits.md` 选取角色的个性、情绪、行为、语言风格
3. 从 `plot-elements.md` 选取事件类型、事件风格、场景氛围
4. 将选取的元素自由组合，确保：
   - 同一角色在不同场景有不同组合（害羞时咬唇、愤怒时毒舌、伤心时沉默）
   - 不同角色之间组合不重复
   - 核心体验维度做到极致
5. 用模块A/B/C的技法将组合落地为具体文字
6. 结合 `extensions/` 中的扩展元素（如有）

### 组合示例

**暧昧章**：
- 角色：高冷型×害羞情绪×咬唇行为×惜字如金语言
- 情节：日常暧昧×心动甜蜜风格×暧昧私密氛围
- 技法：用模块C的感官分层法写亲密，用模块A的反差感塑造法写角色
- 核心体验：暧昧感极致化 → 让读者脸红心跳

**搞笑章**：
- 角色：活泼型×愤怒情绪×跺脚行为×毒舌语言
- 情节：误会冲突×搞笑风格×日常烟火氛围
- 技法：用模块C的对比式幽默，用模块A的表里双层叙事法
- 核心体验：笑点极致化 → 让读者笑出声

**悲伤章**：
- 角色：温柔型×伤心情绪×沉默行为×低声语言
- 情节：离别×心碎风格×雨夜氛围
- 技法：用模块C的重复递进情绪法，用模块B的以死换情法
- 核心体验：悲伤感极致化 → 让读者泪流不止

### 核心体验极端化

**每个内容都要声明核心体验类型，并将其推到极致**：

| 核心体验 | 极端化标准 | 关键技法 |
|----------|-----------|---------|
| 暧昧感 | 让读者脸红心跳，反复回味 | 感官分层法+克制留白法+渐进式隐喻法 |
| 诱惑力 | 让读者血脉贲张，欲罢不能 | 感官分层法+环境烘托法+微反应刻画法 |
| 悲伤感 | 让读者泪流不止，久久不能释怀 | 重复递进情绪法+以死换情法+环境烘托法 |
| 笑点密度 | 让读者笑出声来，停不下来 | 内心独白式幽默+对比式幽默+破折号旁白法 |
| 悬念感 | 让读者猜不到结局，欲罢不能 | 信息差驱动法+钩子设置技巧+嵌套式叙事框架 |
| 热血感 | 让读者热血沸腾，握紧拳头 | 高潮场景设计+重复递进情绪法 |
| 甜蜜度 | 让读者甜到牙疼，嘴角上扬 | 感官叠加法+微反应刻画法+环境烘托法 |
| 治愈感 | 让读者感到温暖，内心平静 | 场景构建+感官叠加法+生活细节法 |

**优势**：通过去耦合化组合+极端化，同一对角色可以有无数种不同的互动方式，每种都能做到极致，避免陷入单一模式。

## 模块概述

### 模块A：角色塑造（~200行）
核心六要素：反差感塑造法、圣体系统、语言风格差异化、行为标签化、家庭背景对比法、互动模式设计。另有14种塑造手法补充。**具体角色类型分类见 character-traits.md**。

### 模块B：情节构造（~270行）
七大板块：主线剧情构造（7种技法）、感情线构造（9种纠缠技法）、钩子设置技巧（9种钩子类型）、冲突设计、节奏控制、高潮场景设计、可复用模板。**具体情节元素分类见 plot-elements.md**。

### 模块C：正文写作（~580行）
八大板块：亲密场景写作（18种技法）、叙事节奏控制、幽默技巧（10种类型）、配角功能性写作、电影化场景写作、对话写作技巧（16种语言风格）、内心独白技巧、场景转换技巧。

### 模块D：反面教材+公式（~230行）
六大烂文模式速查 + 14个写作公式 + 作者元评论方法论 + 综合创作清单。
