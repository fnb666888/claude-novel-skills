# 情节元素分类库（去耦合化）

**核心理念**：事件类型（发生了什么）、事件风格（给读者什么感受）、场景氛围（在什么环境下）是独立维度，可以根据需要自由组合。一个"日常对话"场景可以是"有趣"的，也可以是"虐心"的，还可以是"悬疑紧张"的——组合越丰富，情节越不单调。

**本文件是独立分类库，与构造技法（module-b-plot.md）解耦。** 构造技法告诉你"怎么设计情节"，本文件告诉你"有什么可选"。

---

## 一、事件类型库

事件类型是"发生了什么事"，决定了章节的核心内容和推进方向。

| 事件类型 | 核心任务 | 典型元素 | 适合章节类型 | 字数参考 |
|----------|----------|----------|-------------|----------|
| 日常对话 | 展现关系、推进感情、增加烟火气 | 吃饭、逛街、居家、闲聊、做饭 | 日常章 | 2000-4000 |
| 日常暧昧 | 推进亲密、制造心动、积累张力 | 意外接触、暧昧对话、小动作、眼神交流 | 亲密章 | 2000-4000 |
| 性爱 | 亲密描写、关系确认、情感升华 | 渐进式隐喻、感官分层、余韵、克制留白 | 亲密章 | 3000-5000 |
| 装逼打脸 | 展现实力、制造爽感、确立地位 | 对手膨胀→主角出手→震惊→膜拜 | 高潮章 | 3000-5000 |
| 生死危机 | 制造紧张、推动剧情、展现成长 | 追杀、倒计时、绝境求生、以弱胜强 | 高潮章 | 3000-5000 |
| 身份揭露 | 制造反转、改变预期、推动剧情 | 真实身份曝光、秘密被发现、谎言揭穿 | 转折章 | 2000-4000 |
| 修罗场 | 多人互动、制造冲突、展现关系 | 多角色同场、暗战、选择困境、醋意 | 高潮章 | 3000-5000 |
| 修炼突破 | 展现成长、制造期待、推进力量线 | 闭关、突破、获得新能力、领悟 | 日常章 | 2000-3000 |
| 商业对决 | 展现智慧、制造爽感、推进事业线 | 谈判、竞争、逆转、策略 | 高潮章 | 3000-5000 |
| 离别重逢 | 制造情感高峰、深化关系 | 告别、思念、重逢、意外相遇 | 高潮章 | 2000-4000 |
| 误会冲突 | 制造矛盾、推动剧情、情感爆发 | 误解、越描越黑、真相大白、和好 | 转折章 | 2000-4000 |
| 日常搞笑 | 调节节奏、展现角色魅力 | 沙雕日常、吐槽、互损、误会喜剧 | 日常章 | 2000-3000 |
| 伏笔回收 | 满足期待、制造惊喜、推进主线 | 揭露真相、呼应前文、信息整合 | 转折章 | 2000-4000 |
| 家长互动 | 增加真实感、推动关系、社会压力 | 见家长、家庭聚餐、家长反应、认可/反对 | 日常章 | 2000-4000 |
| 告白/承诺 | 关系确认、情感高峰、里程碑 | 表白、承诺、确认关系、戒指 | 亲密章 | 2000-3000 |
| 保护/牺牲 | 深化羁绊、制造感动、推动关系 | 挡刀、受伤、代替受罚、放弃 | 高潮章 | 2000-4000 |
| 信息揭露 | 推进剧情、制造悬念、改变格局 | 发现秘密、获得情报、真相浮现 | 转折章 | 2000-3000 |
| 挑战/比赛 | 展现实力、制造紧张、推进成长 | 比武、考试、竞赛、对决 | 高潮章 | 3000-5000 |
| 日常陪伴 | 推进感情、增加默契、烟火气 | 一起学习、一起运动、一起旅行 | 日常章 | 2000-3000 |
| 内心独白 | 展现心理、推进情感、增加深度 | 回忆、反思、挣扎、决定 | 日常章 | 1000-2000 |

## 二、事件风格库

事件风格是"给读者什么感受"，决定了章节的情感基调和阅读体验。**同一事件类型可以搭配不同风格**。

| 事件风格 | 核心感受 | 读者反应 | 典型手法 | 适合搭配事件类型 |
|----------|----------|----------|----------|-----------------|
| 有趣/幽默 | 读者笑出来 | "哈哈哈""笑死" | 内心独白、反差萌、误会、旁白吐槽 | 日常对话、日常搞笑、修罗场 |
| 爽点高 | 读者"卧槽" | "太爽了""打得好" | 装逼打脸、实力碾压、逆转、旁观者膜拜 | 装逼打脸、商业对决、挑战/比赛 |
| 反差 | 读者"没想到" | "没想到她是这种人" | 表里不一、身份反转、行为反转、人前人后 | 身份揭露、日常对话、日常暧昧 |
| 热血澎湃 | 读者激动 | "燃起来了""冲！" | 绝地反击、突破极限、以少胜多、觉醒 | 生死危机、修炼突破、挑战/比赛 |
| 心动甜蜜 | 读者心跳加速 | "磕到了""好甜" | 暧昧推拉、亲密升级、意外接触、暧昧对话 | 日常暧昧、性爱、告白/承诺 |
| 虐心 | 读者泪目 | "哭了""好心疼" | 离别、牺牲、误会、错过、隐忍 | 离别重逢、误会冲突、保护/牺牲 |
| 悬疑紧张 | 读者想知道真相 | "然后呢？""谁干的？" | 信息差、线索、伏笔、反转 | 身份揭露、伏笔回收、信息揭露 |
| 温馨治愈 | 读者放松 | "好暖""想要这种生活" | 日常细节、关心照顾、烟火气、相互陪伴 | 日常对话、日常陪伴、家长互动 |
| 禁忌刺激 | 读者"不能这样" | "好刺激""这能写吗" | 身份禁忌、规则禁忌、道德边缘、偷偷摸摸 | 日常暧昧、性爱、修罗场 |
| 荒诞搞笑 | 读者无语笑 | "什么鬼""笑不活了" | 逻辑荒谬、场面失控、旁观者呆滞、误会升级 | 日常搞笑、修罗场、误会冲突 |
| 深情 | 读者感动 | "好深情""在一起" | 等待、守护、默默付出、关键时刻告白 | 告白/承诺、离别重逢、保护/牺牲 |
| 压抑 | 读者紧张 | "好压抑""快爆发" | 沉默、忍耐、压抑、暗流涌动 | 误会冲突、内心独白、生死危机 |
| 逆袭 | 读者期待 | "终于翻身了""爽" | 从低谷崛起、被打压后反击、证明自己 | 装逼打脸、商业对决、挑战/比赛 |
| 暧昧拉扯 | 读者心痒 | "在一起啊""别磨叽" | 推拉、欲擒故纵、差一点、克制 | 日常暧昧、修罗场、告白/承诺 |

## 三、场景氛围库

场景氛围是"在什么环境下"，通过环境要素强化情感基调。**同一事件在不同氛围下效果完全不同**。

| 场景氛围 | 环境要素 | 情感基调 | 适合事件类型 | 适合事件风格 |
|----------|----------|----------|-------------|-------------|
| 暧昧私密 | 灯光昏暗、空间狭小、音乐低沉 | 紧张、心动、暧昧 | 日常暧昧、性爱、告白 | 心动甜蜜、禁忌刺激 |
| 轻松日常 | 阳光、微风、咖啡厅、客厅 | 放松、温馨、自然 | 日常对话、日常搞笑、日常陪伴 | 有趣、温馨治愈 |
| 紧张对峙 | 天色阴沉、气氛凝重、安静 | 紧张、压迫、危险 | 生死危机、商业对决、误会冲突 | 悬疑紧张、压抑 |
| 浪漫仪式 | 烟火、星空、花瓣、烛光 | 感动、郑重、浪漫 | 告白/承诺、离别重逢 | 心动甜蜜、深情 |
| 尴尬喜剧 | 公共场合、意外闯入、有人围观 | 尴尬、搞笑、混乱 | 修罗场、日常搞笑、误会冲突 | 有趣、荒诞搞笑 |
| 孤独回忆 | 雨夜、空房间、旧物、照片 | 思念、伤感、怀旧 | 离别重逢、内心独白、伏笔回收 | 虐心、深情 |
| 战斗热血 | 战场、擂台、风雨、烈火 | 激昂、紧张、热血 | 生死危机、装逼打脸、挑战/比赛 | 热血澎湃、爽点高 |
| 禁忌诱惑 | 夜色、酒气、私密空间、暧昧灯光 | 禁忌、刺激、暧昧 | 性爱、日常暧昧 | 禁忌刺激、心动甜蜜 |
| 校园青春 | 教室、操场、图书馆、放学路 | 青春、纯真、悸动 | 日常对话、日常暧昧、告白 | 心动甜蜜、温馨治愈 |
| 庄严神圣 | 宫殿、宗门大殿、祭坛 | 庄重、肃穆、神圣 | 身份揭露、信息揭露、告白 | 深情、悬疑紧张 |
| 自然山水 | 山顶、湖边、竹林、花海 | 宁静、开阔、诗意 | 日常陪伴、离别重逢、修炼突破 | 温馨治愈、深情 |
| 市井烟火 | 街边小摊、菜市场、夜市 | 热闹、真实、接地气 | 日常对话、日常搞笑 | 有趣、温馨治愈 |
| 雨雪天气 | 下雨、下雪、打雷、起风 | 忧郁、浪漫、压抑 | 离别重逢、误会冲突、告白 | 虐心、深情、心动甜蜜 |

## 四、组合使用指南

### 4.1 组合公式

```
章节内容 = 事件类型 × 事件风格 × 场景氛围 × 角色特点
```

**同一个事件类型，搭配不同风格和氛围，效果完全不同**：

| 事件类型 | 风格 | 氛围 | 效果 |
|----------|------|------|------|
| 日常对话 | 有趣/幽默 | 市井烟火 | 两人在夜市吃小吃，互相吐槽，笑点不断 |
| 日常对话 | 温馨治愈 | 轻松日常 | 两人在家做饭，安静温馨，烟火气十足 |
| 日常对话 | 暧昧拉扯 | 暧昧私密 | 两人在昏暗酒吧，言语试探，暗流涌动 |
| 日常对话 | 虐心 | 孤独回忆 | 两人在雨夜，回忆过去，欲言又止 |
| 性爱 | 心动甜蜜 | 暧昧私密 | 烛光下的温柔亲密，心跳加速 |
| 性爱 | 禁忌刺激 | 禁忌诱惑 | 酒后失控，禁忌感拉满 |
| 性爱 | 深情 | 浪漫仪式 | 烟火下的告白后，情感升华 |
| 装逼打脸 | 爽点高 | 战斗热血 | 擂台上碾压对手，全场震惊 |
| 装逼打脸 | 有趣/幽默 | 尴尬喜剧 | 装逼失败，反被打脸，喜剧效果 |
| 装逼打脸 | 反差 | 紧张对峙 | 看似弱者突然爆发，反转震撼 |

### 4.2 风格混搭技巧

**同一章节可以有多种风格的渐进转换**：

| 章节结构 | 风格转换 | 效果 |
|----------|----------|------|
| 开头→中间→结尾 | 轻松→紧张→爽 | 先放松再制造紧张，最后释放爽感 |
| 开头→中间→结尾 | 甜蜜→虐心→深情 | 先甜后虐，最后感动 |
| 开头→中间→结尾 | 日常→意外→悬疑 | 先日常再突然转折，制造悬念 |
| 开头→中间→结尾 | 压抑→爆发→治愈 | 先压抑再释放，最后温暖 |

### 4.3 避免单一模式

**问题**：情节总是"日常暧昧→亲密→日常"，陷入套路。
**解决**：通过事件类型和风格的组合，打破固定模式。

**示例——同一对角色的多样化情节**：
| 章节 | 事件类型 | 风格 | 氛围 | 效果 |
|------|----------|------|------|------|
| 第5章 | 日常对话 | 有趣 | 市井烟火 | 夜市吃小吃，笑点不断 |
| 第6章 | 日常暧昧 | 心动甜蜜 | 暧昧私密 | 昏暗灯光下的暧昧 |
| 第7章 | 误会冲突 | 虐心 | 雨雪天气 | 雨中争吵，心疼 |
| 第8章 | 装逼打脸 | 爽点高 | 战斗热血 | 为她出头，碾压对手 |
| 第9章 | 离别重逢 | 深情 | 浪漫仪式 | 重逢告白，感动 |
| 第10章 | 日常搞笑 | 荒诞搞笑 | 尴尬喜剧 | 修罗场，笑不活了 |
| 第11章 | 生死危机 | 热血澎湃 | 战斗热血 | 绝地反击，燃 |
| 第12章 | 性爱 | 禁忌刺激 | 禁忌诱惑 | 酒后失控 |

### 4.4 事件类型×角色特点联动

**同一事件类型，不同个性角色的参与方式不同**：

| 事件类型 | 高冷角色 | 活泼角色 | 腹黑角色 | 傲娇角色 |
|----------|----------|----------|----------|----------|
| 日常对话 | 惜字如金但认真听 | 话多、跑题、兴奋 | 暗戳戳试探 | 嘴硬但在意 |
| 装逼打脸 | 冷脸碾压，不屑解释 | 兴奋炫耀，嘚瑟 | 暗中布局，事后装无辜 | 嘴上说"不关我事"但尾巴翘 |
| 修罗场 | 冷眼旁观，不参与 | 兴奋看戏，火上浇油 | 暗中挑拨，坐收渔利 | 嘴硬说"我才不在意" |
| 生死危机 | 冷静分析，果断行动 | 慌张但不放弃 | 算计逃生路线 | 嘴硬说"谁要你保护" |

## 五、常见情节模式速查

以下是一些常见的情节模式，由事件类型+风格+氛围组合而成：

### 5.1 甜宠模式
- 事件类型：日常暧昧 + 日常对话
- 事件风格：心动甜蜜 + 温馨治愈
- 场景氛围：轻松日常 + 暧昧私密
- 适合：感情线推进、日常章节

### 5.2 爽文模式
- 事件类型：装逼打脸 + 商业对决
- 事件风格：爽点高 + 逆袭
- 场景氛围：战斗热血 + 紧张对峙
- 适合：高潮章节、事业线推进

### 5.3 虐心模式
- 事件类型：离别重逢 + 误会冲突
- 事件风格：虐心 + 深情
- 场景氛围：雨雪天气 + 孤独回忆
- 适合：情感高潮、关系转折

### 5.4 悬疑模式
- 事件类型：身份揭露 + 信息揭露 + 伏笔回收
- 事件风格：悬疑紧张 + 反差
- 场景氛围：紧张对峙 + 孤独回忆
- 适合：剧情转折、主线推进

### 5.5 搞笑模式
- 事件类型：日常搞笑 + 修罗场
- 事件风格：有趣/幽默 + 荒诞搞笑
- 场景氛围：尴尬喜剧 + 轻松日常
- 适合：节奏调节、减压阀

### 5.6 禁忌模式
- 事件类型：日常暧昧 + 性爱
- 事件风格：禁忌刺激 + 心动甜蜜
- 场景氛围：禁忌诱惑 + 暧昧私密
- 适合：亲密章节、感情线高潮

### 5.7 热血模式
- 事件类型：生死危机 + 挑战/比赛 + 修炼突破
- 事件风格：热血澎湃 + 爽点高
- 场景氛围：战斗热血
- 适合：高潮章节、成长线推进

### 5.8 治愈模式
- 事件类型：日常对话 + 日常陪伴 + 家长互动
- 事件风格：温馨治愈 + 有趣/幽默
- 场景氛围：轻松日常 + 市井烟火 + 自然山水
- 适合：日常章节、节奏舒缓
