# 模块D：反面教材——常见烂文模式

**知道什么不该写，和知道什么该写同样重要。**

## 一、人设崩塌的典型模式

| 崩塌类型 | 表现 | 如何避免 |
|----------|------|----------|
| **前后矛盾** | 前文说"冷酷无情"，后文突然"多愁善感" | 建立角色档案，每次写前review |
| **能力膨胀** | 主角无限制变强，没有代价 | 金手指必须有明确限制 |
| **性格工具化** | 角色性格随剧情需要变化 | 角色行为必须符合已建立的性格逻辑 |
| **标签化** | 角色只有一个标签，没有深度 | 用"反差感塑造法"增加层次 |
| **失忆式推进** | 角色突然忘记前文发生的事 | 维护角色状态追踪表 |

## 二、情节注水的常见手法

| 注水类型 | 表现 | 如何避免 |
|----------|------|----------|
| **无意义对话** | 对话不推进剧情、不塑造角色 | 每段对话必须有至少一个功能 |
| **重复描写** | 同样的场景/情绪反复描写 | 检查是否有"说过的话再说一遍" |
| **升级打怪循环** | 打完小怪打大怪，没有情感深度 | 每次冲突都要有情感/关系层面的变化 |
| **无用支线** | 支线对主线没有贡献 | 支线必须与主线有交汇点 |
| **强行凑字数** | 环境描写过长、内心独白过多 | 每段描写都要问"删掉会影响故事吗？" |

## 三、亲密描写的雷区

| 雷区 | 表现 | 正确做法 |
|------|------|----------|
| **直白粗俗** | 直接描写肉体，没有美感 | 用隐喻、感官、环境烘托 |
| **千篇一律** | 所有亲密场景写法相同 | 每个角色有不同的亲密模式 |
| **没有情感铺垫** | 突然亲密，没有前因 | 亲密必须有渐进式铺垫 |
| **没有余韵** | 亲密后戛然而止 | 亲密后的余韵比亲密本身更重要 |
| **道德负担过重** | 亲密场景让人有负罪感 | 给亲密一个"合理借口"（中毒/功法/意外） |
| **尺度失控** | 过于露骨，失去想象空间 | "半露不露"比全露更有张力 |

## 四、幽默失败的案例

| 失败类型 | 表现 | 正确做法 |
|----------|------|----------|
| **强行搞笑** | 笑点生硬，读者笑不出来 | 幽默要自然融入场景 |
| **低俗梗** | 用低俗内容博笑点 | 用反差、误会、逻辑包装 |
| **打断节奏** | 在紧张场景插入不合适的幽默 | 幽默要选对时机（减压阀，不是破坏者） |
| **角色失真** | 严肃角色突然变搞笑 | 幽默要符合角色性格 |
| **重复梗** | 同一个笑点反复使用 | 金句可以重复，但笑点要不断更新 |

## 五、节奏控制的常见错误

| 错误 | 表现 | 正确做法 |
|------|------|----------|
| **全程高压** | 一直紧张，没有喘息 | 高压段后必须有低压段 |
| **全程低压** | 一直平淡，没有高潮 | 每5-8章必须有一个小高潮 |
| **节奏突变** | 从紧张突然变平淡，没有过渡 | 用"减压阀"自然过渡 |
| **没有钩子** | 章末没有悬念，读者不想翻页 | 每章结尾必须有钩子 |
| **高潮过多** | 每章都是高潮，读者疲劳 | 高潮要稀疏分布，才有冲击力 |

## 六、世界观的常见硬伤

| 硬伤 | 表现 | 如何避免 |
|------|------|----------|
| **等级体系混乱** | 等级划分不清晰，战力崩塌 | 建立明确的等级体系表 |
| **规则随意打破** | 已建立的规则被主角随意违反 | 规则必须有例外的合理解释 |
| **地理矛盾** | 距离/方位前后不一致 | 维护地理设定文档 |
| **时间线混乱** | 事件顺序不合理 | 维护时间线追踪表 |
| **势力关系矛盾** | 势力间关系前后不一致 | 维护势力关系图 |

## 附录：九大写作公式

### 公式一：亲密场景公式
暧昧氛围铺垫 + 感官细节递进 + 角色性格反应 + 隐喻收尾

### 公式一补充：亲密场景进阶公式
- **渐进隐喻公式**：小雨（试探）→ 中雨（升温）→ 大雨（高潮）→ 雷鸣（爆发）——读者看到信号就知道升级
- **禁忌张力公式**：设定绝对禁忌 → 边缘反复试探 → "合理借口"突破 → 禁忌破除的爽感
- **克制留白公式**：亲密升温 → 即将到达顶点 → 角色/外力打断 → 读者心痒 → 下次突破
- **能力替代公式**：世界观设定（灵力/功法/毒）驱动亲密 → 亲热=剧情需要 → 能力流动映射快感
- **喜剧打断公式**：亲密升温 → 第三人闯入/误会/功法反噬 → 尴尬喜剧 → 收拾残局（比亲密本身更有趣）

### 公式二：商业爽感公式
对手膨胀 → 主角出手 → 对手震惊 → 旁观者膜拜

### 公式三：幽默公式
正常铺垫 + 意外转折 + 角色特色爆发

### 公式四：情感铺垫公式
日常细节积累 + 信息不对称 + 情绪临界点爆发 + 余韵留白

### 公式五：节奏控制公式
快节奏（商业/冲突）→ 减压阀（日常/搞笑）→ 慢节奏（情感/铺垫）→ 高潮爆发

### 公式六：驯绿茶公式
大棒（严厉训斥/冷暴力）→ 情绪低谷（哭泣/道歉/恐惧）→ 甜枣（突然的礼物/温柔/承诺）→ 情绪剧烈波动

### 公式七：商业舆论战公式
暗中布局 → 等待对手犯错 → 高调宣布 → 收割胜利果实

### 公式八：信息套取公式
假装不知 → 层层铺垫 → 引导对方主动透露 → 收网

### 公式九：误会递进公式
初始误会 → 越描越黑 → 误会升级 → 真相揭晓（一次性反转，喜剧释放）

### 公式十：以死换情公式（来自《仙子》）
建立深厚羁绊 → 情感最高点牺牲 → 递进式告别台词 → 死亡浪漫化 → 继承/觉醒

### 公式十一：亲吻感官对比公式（来自《青葱》）
亲吻作为"中段节拍" → 不同角色不同唇感+气味 → 用隐喻区分体验 → 继续推进场景

### 公式十二：三步递进亲密公式（来自《仙子》）
暗示（微表情+气息）→ 接触（第一次身体碰触）→ 心理反馈（内心感受）→ 每步独立段落

### 公式十三：逻辑包装撩拨公式（来自《青葱》）
角色想亲密 → 用一本正经的歪理包装 → 对方无法反驳 → 亲密行为"合理化"

### 公式十四：数据揭示公式（来自《青葱》）
铺垫期待 → 破折号+具体数字 → 数字本身成为高潮 → 角色/读者同时震惊

## 作者元评论中的方法论

### 1. 感情线与商业线的比例
感情线是主轴，商业线是支撑，两者比例约为6:4。

### 2. "明明白白的渣"与"暗渣"
选择"明明白白的渣"——主角从不否认，但通过能力、魅力和对每个角色的"真心"来让读者接受。

### 3. 时间流速与情感线的关系
时间流速跟着感情线走——关键阶段时间变慢，平淡期时间加速。

### 4. 配角的"功能性"设计
每个配角在出场前就已经被赋予了明确的"叙事功能"，是"需要时才召唤的"。

### 5. 金手指必须有代价（来自《仙子》+《青葱》）
任何超能力/先知能力都必须有明确的限制和成本，否则冲突无法成立。

- **《仙子》的圣体代价**：先天剑骨被强行抽离、七窍玲珑心成为被追杀的目标、阴阳眼可以被当药引
- **《青葱》的记忆宫殿代价**：过度使用导致眩晕和脑力疲劳、印象越深消耗越少（不是全能）、无法复制整本书

### 6. 感官差异化是亲密场景的灵魂（来自《青葱》）
不同角色的亲密体验必须有完全不同的"味道"，否则所有感情线都会变得雷同。

- 应禅溪=棉花糖+溪水般温顺
- 颜竹笙=薄荷+抓雪入衣服的刺激
- 徐有渔=温热肌肤+成熟丰满

- [ ] 设计钩子类型（悬念/转折/信息揭露）
