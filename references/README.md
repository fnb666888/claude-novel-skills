# 参考资料目录

将你认为优秀的小说文件放在此目录下，系统会在创作时自动参考这些作品进行对比优化。

## 使用方式

1. 将优秀小说的文件放入此目录（支持 `.txt` 和 `.md` 格式）
2. 文件命名建议：`[小说名]_[作者].txt` 或 `[小说名]_[作者].md`
3. 系统会自动读取此目录下的所有小说文件作为参考

## 对比优化流程

当创作灵感、大纲或正文时，系统会：

1. **读取参考资料**：扫描此目录下的所有小说文件
2. **结构对比**：对比叙事结构、节奏控制、伏笔设置等
3. **内容对比**：对比场景描写、人物塑造、对话写作等
4. **技法对比**：对比钩子设置、情感渲染、氛围营造等
5. **定向优化**：根据对比结果，将生成内容优化到参考作品的水平

## 建议收录

- 同类型的优秀作品（便于对比学习）
- 不同风格的标杆作品（拓宽创作思路）
- 你特别喜欢的某些章节或片段

## 注意事项

- 文件越大，分析越全面，但处理时间也越长
- 建议收录完整小说或至少包含关键章节的文件
- 系统会自动识别文件格式（txt/md）
