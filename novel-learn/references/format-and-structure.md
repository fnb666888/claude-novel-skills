# 导入格式规范

## 正文文件格式

### 章节文件命名
- 格式：`第XXX章_章名.md`
- 章节号用 3 位数字补零：001, 002, ..., 999
- 章名与原文一致（去除特殊字符）

### 章节文件内容
```markdown
## 第X章 章名

正文内容...
```

### 格式要求
- 一句一段
- 段落间不加空行
- 对话独立成行，用半角双引号
- 不使用对话标签（"说道/问道"）
- 章节标题用 `##` 格式
