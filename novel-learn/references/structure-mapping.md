# 结构映射规则

## 章节切分规则

### 常见分隔符模式

| 模式 | 正则 | 说明 |
|------|------|------|
| 第X章 | `第[一二三四五六七八九十百千\d]+章` | 最常见 |
| 第X节 | `第[一二三四五六七八九十百千\d]+节` | 较常见 |
| 第X卷 | `第[一二三四五六七八九十百千\d]+卷` | 卷分隔 |
| Chapter X | `Chapter\s+\d+` | 英文 |
| 数字开头 | `^\d+\s*[\.\、]` | 简单编号 |

### 切分策略

1. 优先尝试 `第X章` 模式
2. 如果命中率 < 50%，尝试其他模式
3. 如果所有模式都失败，按固定字数切分（每 3000-5000 字一段）
4. 用户可手动指定分隔符

### 章名提取

- 从分隔符后的同一行提取章名
- 去除多余空格和标点
- 如果无章名，使用 "第X章" 作为默认名

```
└── ...

└── world_[书名].md [导入反推]
追踪/
├── 伏笔.md [导入反推]
└── 进度.md
```
