# 角色状态逆向提取

## 提取维度

### 1. 基本信息
- 姓名/称谓
- 年龄（如有明确提及）
- 身份/职业
- 外貌特征（2-3 个标志性特征）

### 2. 性格特征
- 公众形象（他人眼中的她/他）
- 私人形象（亲密关系中的她/他）
- 隐藏形象（只有读者知道的她/他）
- 核心反差点

### 3. 语言风格
- 口头禅/标志性句式
- 说话节奏（快/慢/断续）
- 语气特征（软萌/高冷/痞气/...）

### 4. 关系网络
- 与主角的关系类型
- 关系发展轨迹
- 关键互动事件

### 5. 能力/状态
- 当前能力水平
- 重要装备/道具
- 身体状态（如有特殊状态）

## 提取方法

1. Grep 角色名，统计出场次数
2. 提取每次出场的关键描写
3. 对比不同章节，识别性格一致性
4. 提取对话样本，分析语言风格
5. 整理关系发展时间线

## 输出格式

```markdown
# [角色名] [导入反推]

## 基本信息
- 姓名：
- 年龄：
- 身份：
- 外貌：

## 性格特征
- 公众形象：
- 私人形象：
- 核心反差：

## 语言风格
- 口头禅：
- 说话节奏：
- 语气特征：

## 关系网络
- 与主角：
- 关键事件：

## 备注
- 数据来源：[章节号]
- 需要复核：是/否
```
