# 去AI味规则集

## 6 Gate 门禁系统

| Gate | 检测维度 | 说明 |
|------|----------|------|
| **Gate A** | 禁用词 | 扫描 banned-words.md 中的禁用词和句式 |
| **Gate B** | 句式套路 | "不是A而是B"、万能状语、连续排比 |
| **Gate C** | 情绪告知 | "他感到/她意识到" → 用身体反应展示 |
| **Gate D** | 节奏均匀 | 排比、长句、段落长度过于一致 |
| **Gate E** | 对话同质 | 所有角色同一语气、对话标签过多 |
| **Gate F** | 结尾升华 | "他终于明白了……"式总结收尾 |

## 量化分级（6 指标客观评分）

| 指标 | 轻度 | 中度 | 重度 |
|------|------|------|------|
| 禁用词密度（每千字命中） | ≤5 | 6-15 | >15 |
| 连续排比数 | ≤2 | 3-4 | ≥5 |
| 心理描写词比例 | ≤10% | 10-25% | >25% |
| 对话标签密度 | ≤30% | 30-50% | >50% |
| 平均段落句数 | ≤3 | 3-5 | >5 |
| 重复描写密度（每千字） | ≤1 | 2-3 | ≥4 |

**最高严重度决定整体级别。**

## 三遍修复法

| 级别 | 策略 |
|------|------|
| 轻度 | Pass 1：去泛化 |
| 中度 | Pass 1 + Pass 2：去泛化 + 去书面化 |
| 重度 | 完整三遍 + 重点段落重写 |

## Gate A：禁用词扫描

**完整禁用词表**：`[skill]/rules/banned-words.md`

Gate A 扫描时读取 `banned-words.md`，包含：
- 最毒禁用句式（★★★★★ 至 ★★，含错误例和修法）
- 一级禁用词（出现即替换）
- 二级禁用词（高频出现时替换）

## Gate B：句式去套路

检测模式：
- "不是A，而是B" / "不是A，不是B，而是C"
- "...，带着..." 万能状语
- 连续 3 句以上相同结构的排比
- "有的...有的...有的..."
- 工整对比句式

## Gate C：情绪告知检测

检测模式：
- "他感到/她感到/他觉得/她觉得"
- "他意识到/她意识到"
- "他终于明白/她终于明白"
- 直接写情绪词：悲伤、愤怒、恐惧、紧张、失落

## Gate D：节奏均匀检测

检测模式：
- 连续 5 段以上长度相近（±5 字）
- 连续 3 句以上句式结构相同
- 段落内逗号超过 5 个
- **连续 5 句以上都是短句（<10字）** → 合并或添加长句
- **连续 5 句以上都是长句（>20字）** → 拆分或添加短句
- **一句话一段** → 合并到相邻段落

## Gate E：对话同质检测

检测模式：
- 对话标签密度 > 30%（"说道/问道/叹道"）
- 所有对话句式长度相近
- 缺少口语化表达

## Gate F：结尾升华检测

检测模式：
- "这一刻..."
- "他/她终于明白了..."
- "原来..."
- "一切...都..."
- "他不知道的是..."
- "这一夜，注定..."

## 保护机制

### 白名单（`.deslop-whitelist`）

项目根目录的 `.deslop-whitelist` 文件中的词汇不会被标记为禁用词。

### 删除比例上限

| 级别 | 最大删除比例 |
|------|-------------|
| 轻度 | ≤15% |
| 中度 | ≤25% |
| 重度 | ≤35% |

### 收敛停止

- 同一段落连续 2 轮无新变化 → 停止该段修复
- 全文最多 3 轮完整修复

### 不可删除

- **禁止整段删除正文内容**
- 禁止删除情节信息、设定信息、角色关系
- 只能将信息改写为更自然的表达
