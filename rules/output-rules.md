# 输出文档规则

**重要：每次运行本skill后，必须将生成的创作内容保存为文档。**

## 文件系统分类

```
~/小说/
├── 灵感/                              # 灵感最终版
│   └── 灵感_[小说名]_最终版.md
│
├── 大纲/                              # 大纲相关
│   ├── 大纲_[小说名]_最终版.md         # 全书大纲最终版
│   ├── 角色系统_[小说名].md            # 角色系统
│   └── 章节大纲/                       # 章节大纲
│       ├── 章节大纲_[小说名]_最终版.md
│       ├── 世界线系统_[小说名].md
│       └── 角色剧情_[小说名].md
│
├── 正文/                              # 正文（纯txt，无评价内容）
│   └── [小说名]/
│       ├── 第001章_[章节名].txt
│       ├── 第002章_[章节名].txt
│       └── ...
│
├── 设定/                              # 世界设定（灵感阶段创建）
│   └── world_[小说名].md
│
└── 追踪/                              # 写作反馈（自动）
    └── 写作反馈.md
```

## 输出格式规则

| 内容类型 | 格式 | 说明 |
|----------|------|------|
| 灵感 | `.md` | 只输出最终版 |
| 大纲（全书） | `.md` | 最终版 + 角色系统 |
| 章节大纲 | `.md` | 最终版 + 世界线系统 + 角色剧情 |
| 正文 | `.txt` | 纯正文，不含任何评价、优化、注释内容 |
| 角色系统 | `.md` | 角色档案 |
| 世界线系统 | `.md` | 世界线进程追踪 |
| 设定 | `.md` | 世界观文档 |

## 角色系统模板

每个小说维护一个角色系统文件 `角色系统_[小说名].md`，记录所有角色的状态追踪。

```markdown
# 角色系统 — [小说名]

## 角色列表

### [角色名]

**核心特点**：[3词定义，如：高冷/外冷内热/护短]

**当前状态**：
- 身份：[当前身份]
- 能力：[当前能力水平]
- 情感状态：[对谁/什么感觉]

**关系网络**：
- [角色B]：[关系描述] → [变化趋势]
- [角色C]：[关系描述] → [变化趋势]

**成长弧线**：
- 起点：[初始状态]
- 转折1：[第X章，发生了什么，导致什么变化]
- 转折2：[...]
- 当前：[现在处于弧线的什么位置]

**故事线进展**：
- [第X章]：[关键事件]
- [第X章]：[关键事件]
```

## 世界线系统模板

章节大纲阶段维护世界线系统文件 `世界线系统_[小说名].md`，追踪世界进程。

```markdown
# 世界线系统 — [小说名]

## 当前时间点
[小说当前进度，如：第X章 / 故事内第X天]

## 已发生事件

| 章节 | 事件 | 影响 | 关联角色 |
|------|------|------|----------|
| 第X章 | [事件描述] | [对世界/剧情的影响] | [涉及角色] |

## 伏笔状态

| 伏笔 | 埋设章节 | 状态 | 预计回收 |
|------|----------|------|----------|
| [伏笔描述] | 第X章 | 已埋/进行中/已收 | 第X章 |

## 关键转折点

1. [转折描述] — 第X章
2. [转折描述] — 第X章

## 世界状态摘要
[当前世界的整体状态，如势力格局、危机程度等]
```

## 角色剧情模板

章节大纲阶段维护角色剧情文件 `角色剧情_[小说名].md`，记录每个角色在各章节的具体剧情。

```markdown
# 角色剧情 — [小说名]

## [角色名]

| 章节 | 剧情 | 情感变化 | 成长/变化 |
|------|------|----------|-----------|
| 第X章 | [本章该角色的具体剧情] | [情感变化] | [成长或状态变化] |

## [角色名]
...
```

## 输出确认

保存文件后，向用户报告：
- 文件保存路径
- 生成内容的核心要点概述
- 下一步建议
